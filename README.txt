OVERVIEW:
VangtiChai is an Android app built using Flutter. It allows the user to enter
a money amount using a custom numeric keypad and calculates the change in
terms of Taka notes (500, 100, 50, 20, 10, 5, 2, 1).

FEATURES:
- Built a custom numpad from scratch using Flutter layout widgets
- Real time change calculation as digits are entered
- Portrait and landscape layout support, but unable to test the feature. 
- Works on both phones and tablets, unable to test this. 
- All sizes are defined in sizes.dart, no hardcoded values.



DEVICE AND SCREEN TYPES TESTED:

Chrome Browser (Desktop) - Responsive testing
Result: Layout adapts correctly to different window sizes
